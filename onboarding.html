Welcome to Fayara!

Let's get you set up step-by-step. Please check the box when you've completed each action.

Step 1: Install Ollama

Fayara connects to Ollama to run AI models locally.

Download & Install Ollama

If you haven't already, download and install Ollama for your system.

Download Ollama

After downloading, run the installer and follow the on-screen instructions.

Run Ollama

Ensure Ollama is running in the background. You might see an icon in your system tray or menu bar.

Fayara will try to connect to Ollama at http://127.0.0.1:11434 by default.

Step 2: Download Required AI Models

Ollama needs specific models. Use the command line (Terminal/CMD/PowerShell) to download them.

Download Chat Model (Llama 3)

This model is great for general chat and tasks like proofreading.

ollama pull llama3

Model downloads can take time. Find others on the Ollama Library.

Download Embedding Model (nomic-embed-text)

This model is needed for Fayara to understand webpages/documents (RAG feature).

ollama pull nomic-embed-text

Required for "Chat with Website" or "Chat with PDF".

Step 3: Configure Fayara Settings (Optional)

Tell Fayara which models to use in the Settings panel (⚙️ icon).

Review Settings

Briefly review where to set models for different features:

  1. Chat: Selected in the chat dropdown.
  2. Proofread: Settings > Proofread section.
  3. Autocomplete: Settings > Autocomplete section.
  4. RAG & Embeddings: Settings > RAG & Embeddings section (uses embedding model).
  5. Click "Save Settings" when done configuring.

You can always configure these settings later.

What Fayara Can Do For You

You're all set! Here's a glimpse of Fayara's key features:

💬 Chat with AI

Engage in conversations, ask questions, get summaries, and more using powerful local AI models.

  • Type your queries directly in the chat window.
  • Ask for explanations, summaries, or creative ideas.
  • Get help with coding, writing, and problem-solving.

📄 Chat with Websites

Understand the content of the current webpage without leaving your browser.

  • Click the "Read Page" icon (👁️) in Fayara's sidebar.
  • Instantly summarize long articles or documents.
  • Ask specific questions about the page's content.
  • Extract key information without manual searching.

📚 Chat with Local PDFs

Ask questions about your PDF documents directly within Fayara.

  • Drag and drop a PDF file onto the Fayara chat window.
  • Securely analyze your PDFs without uploading them.
  • Find information quickly within large documents.
  • Ideal for research papers, reports, and ebooks.

✍️ Autocomplete Text

Get AI-powered suggestions as you type in various text fields across the web (if enabled).

  • Enable Autocomplete in Fayara's settings.
  • Start typing in a text field on any webpage.
  • Press your configured trigger key (default: Tab) for suggestions.
  • Speed up writing emails, posts, and notes.

✅ Proofread Selected Text

Quickly check and correct grammar or rephrase selected text on any webpage.

  • Select any text on a webpage.
  • A context menu will appear with Fayara actions:
  • Proofread: Correct grammar and spelling.
  • Elaborate/Summarize: Expand or condense text.
  • Custom Prompt: Apply your own instructions.
